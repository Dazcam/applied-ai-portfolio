# Applied AI Portfolio

A portfolio of applied AI projects built to demonstrate practical skills in modern AI techniques including RAG pipelines, agent-based approaches, and AI-powered analytics tools.

Each project is framed around a realistic Scottish SME use case, reflecting the kind of short, high-impact engagements typical in applied AI consultancy.

## Projects

### 01 — RAG Pipeline: Document Q&A for a Scottish Food & Drink SME
A retrieval-augmented generation pipeline that allows staff to query internal compliance, supplier, and HR documents using plain English. Built with LangChain, ChromaDB, and Llama 3.3 via Groq.
